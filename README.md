# Wallet Tracker

A professional cryptocurrency portfolio tracker that monitors successful and unsuccessful coin investments with developer attribution.

## Features

- Track multiple cryptocurrency investments
- Categorize coins as successful or unsuccessful
- Record developer names for each coin
- View real-time profit/loss calculations
- Portfolio statistics dashboard
- Edit and update coin information
- Persistent local storage

## Framework

Vite + React 19

## Theme

Clean Light theme with soft shadows, neutral color palette (white/gray backgrounds), and clear visual hierarchy. Cards feature rounded corners and subtle borders for a modern, professional appearance.
